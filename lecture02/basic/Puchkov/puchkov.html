    <input>

    • type="button" – Кнопка с надписью.
    • type="checkbox" – Флажок.
    • type="color" – Виджет для выбора цвета.
    • type="date" – Поле для ввода календарной даты (год, месяц, день).
    • type="datetime" – Поле для ввода даты и времени.
    • type="datetime"-local – Поле для ввода даты и местного времени.
    • type="email" – Поле для ввода адреса электронной почты.
    • type="file" – Поле для ввода имени файла, который пересылается на сервер.
    • type="hidden" – Скрытое поле (не отображается на Web-странице).
    • type="image" – Графическая кнопка отправки данных формы на сервер. Вместе с этим значением нужно использовать атрибут src, чтобы определить адрес изображения и атрибут alt для определения альтернативного текста. Можно также задать атрибуты width и height, чтобы определить размер изображения в пикселях.
    • type="month" – Поле для ввода месяца и года.
    • type="number" – Поле для ввода чисел.
    • type="password" – Поле для ввода пароля.
    • type="radio" – Переключатели (радио-кнопки).
    • type="range" – Ползунок для ввода чисел в указанном диапазоне.
    • type="reset" – Кнопка сброса данных формы в первоначальное состояние.
    • type="search" – Поле для ввода строки поиска.
    • type="submit" – Кнопка для отправки данных формы на сервер.
    • type="tel" – Поле для ввода номера телефона.
    • type="text" – Текстовое поле.
    • type="time" – Поле для ввода времени.
    • type="url" – Поле для ввода Web-адреса.
    • type="week" – Поле для ввода номера недели и года.
